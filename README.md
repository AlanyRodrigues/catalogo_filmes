# Catálogo de Filmes

Projeto de catálogo de filmes desenvolvido em Ruby on Rails, com funcionalidades de cadastro, 
edição, remoção e comentários de filmes.

## Tecnologias utilizadas
- Ruby 3.2.2
- Rails 7.2
- PostgreSQL
